# 🛡️ Phishing Detection System

A Machine Learning-based phishing detection system that identifies whether a website or URL is legitimate or malicious using extracted URL and website features.

## 📌 Overview

Phishing attacks are one of the most common cybersecurity threats, where attackers create fake websites to steal sensitive information such as usernames, passwords, and financial details.

This project leverages Machine Learning techniques to classify URLs as **Phishing** or **Legitimate**, helping users identify potentially dangerous websites before interacting with them.

---

## 🎯 Objectives

- Detect phishing websites using Machine Learning.
- Improve cybersecurity awareness through automated URL analysis.
- Build an accurate and scalable phishing detection model.
- Reduce false positives while maintaining high detection performance.

## 📊 Dataset

The model is trained on a phishing URL dataset containing both legitimate and phishing websites.

Typical features include:

- URL Length
- Number of Dots
- Number of Hyphens
- Number of Digits
- Presence of HTTPS
- Special Characters
- Domain Information
- Suspicious Keywords
